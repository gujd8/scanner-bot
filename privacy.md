# Privacy Policy for Black Night Scanner

**Last Updated: June 2026**

Black Night scanner is committed to protecting user privacy. This policy explains how we handle data within Discord.

### 1. Data Processing
* **Nicknames & Names:** The bot exclusively fetches official usernames from Roblox to update and synchronize user nicknames within the Discord server. It also scans the active server member list in real-time to verify that a requested identity is not already taken before completing the change.
* **No Off-Platform Storage:** We do not store, log, or save any user activities, presence data, or message histories on external databases or off-platform servers. All processing occurs temporarily during active execution.

### 2. Platform Alignment
* Users can manage their platform visibility and settings via Discord's native privacy controls.
