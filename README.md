# Privacy Policy for Black Night scanner bot

Effective Date: June 27, 2026

We take your privacy seriously. This Privacy Policy explains what data our Discord application (Black Night scanner) collects, how it is used, and how it is protected.

## 1. Data We Collect
Our bot only accesses and processes data necessary for its core functionalities:
* **Server Members & Nicknames:** We process user server nicknames and member listings to manage and verify Roblox-related identity formatting.
* **Discord User IDs:** Used temporarily to verify and map your identity with external Roblox database requests.

## 2. How We Use Data
The data collected is used strictly to:
* Verify user identities in connection to Roblox.
* Manage, check, and update user nicknames inside the Discord server to avoid duplication.
* We do not store your message content or store persistent personal tracking logs.

## 3. Data Retention and Sharing
* We do not sell, rent, or share your data with any third parties.
* Data processed during execution is kept only alos ng as necessary to fulfill the operational functions of the bot inside the server.

## 4. Contact Us
If you have any questions regarding this Privacy Policy, feel free to contact the developer via our official Discord Support Server or through the developer profile.
